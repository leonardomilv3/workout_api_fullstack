# Workout API

## Tecnologias

- Backend:
	- Node
	- Empress

- Frontend:
	- React

## Comandos uteis

### Inicializar projeto

Na pasta com index.js, inicialize o projeto com o comando:

- `npm init`

### Istalar uma dependencia

Na pasta com `index.js`, instale uma dependencia com o comando:

- `npm install nome_da_dependencia`
